# Number Guessing Game 🎯

## Project Description
This is a simple Python project where the computer randomly selects a number between 1 and 100.  
The user has to guess the number. The program gives hints if the guess is too high or too low until the correct number is guessed.

## Features
- Random number generation
- User input handling
- Hint system (Too high / Too low)
- Counts number of attempts

## Technologies Used
- Python
- Random module
